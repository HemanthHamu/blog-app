What is node js?

Node.js is a JavaScript runtime environment that is open-source and cross-platform, enabling the execution of JavaScript code outside the confines of a web browser. It allows developers to use JavaScript to write server-side scripts, enabling the development of scalable and high-performance web applications. Node.js is built on the V8 JavaScript runtime engine, which is developed by Google for use in the Chrome browser. Key features of Node.js include: Non-blocking, Event-Driven Architecture: Node.js is designed to be non-blocking and event-driven, making it well-suited for handling concurrent connections and building scalable applications. It uses an event loop to efficiently manage I/O operations.




What is mongoDB?

MongoDB is a type of database that stores information in a flexible, JSON-like format. It's good for handling lots of data and can adapt to changing structures. Unlike traditional databases, it doesn't require a fixed plan for how data should look. MongoDB is fast, can grow easily, and copies data across servers for reliability.

How to connect mongoDB to JavaScript?

To connect mongoDB to JavaScript we have to use a package called mongoose.

Mongoose acts as a go-between, connecting your code with MongoDB through a well-organized JavaScript library. It serves as a translator, streamlining the interaction with databases and improving the ease of managing data.

Difference between mongoDB and mongoose

mongoDB is a database where you can see the data that is collected from various sources like login form,registration form etc..

mongoose is used to design the database in the way you want.
